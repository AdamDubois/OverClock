# Projet Bombe - Altium Designer

## Description
Vous trouverez ici tous l'ensemble des fichiers liés à la conception électronique réalisée avec la plateforme Altium Designer 

Vous trouverez les schémas, les bibliothèques de composants, et les deux conceptions pour les cartes électroniques du projet.

## Structure du dossier
- Facture - contiens les factures reliées à la conception des cartes électroniques.
- Bibliothèques - regroupe les bibliothèques des composantes utilisées dans le projet.
- Carte de connecteur - tous les fichiers pour la conception de la carte électronique principale du projet.
- Carte RFID - contiens les fichiers pour la conception de la carte électronique pour le contrôle des modules RFID.
