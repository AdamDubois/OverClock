# Projet Bombe - Documentation/Référence/Notes
## Description
Ce dossier contient l'ensemble des notes utilisées pour la réalisation du projet Bombe. Ces notes incluent des idées, des réflexions, des observations et des informations pertinentes pour la réalisation du projet. 

## Architecture de ce dossier
Ce dossier est organisé de la manière suivante :
- Budget de puissance: une estimation de la consommation électrique de chaque composant du projet, afin de s'assurer que l'alimentation est suffisante pour faire fonctionner tous les éléments.
