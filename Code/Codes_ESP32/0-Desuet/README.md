# Projet Bombe - Code/Codes_ESP32/0-Desuet

## Description

Ce dossier contient tous les projets de code qui ne sont plus utilisés ou qui sont obsolètes. Ces projets ont été créés avec un autre ESP comme base. Ils sont conservés à des fins de référence, mais ne sont plus maintenus ni utilisés dans le projet actuel. Ces projets peuvent inclure des versions antérieures du code pour les ESP32, des prototypes ou des expérimentations qui ont été abandonnés au profit de solutions plus efficaces ou mieux adaptées aux besoins du projet Bombe.

## Architecture de ce dossier

Ce dossier contient les fichiers de code pour les différents composants du projet Bombe qui sont obsolètes ou ne sont plus utilisés. Il est organisé de la manière suivante :
- `Desuet_ESP32_Feather/` : Contient tous les fichiers de code qui ont été développés pour un ESP32 Feather. Ces fichiers sont obsolètes et ne sont plus utilisés dans le projet actuel, mais ils sont conservés à des fins de référence.
