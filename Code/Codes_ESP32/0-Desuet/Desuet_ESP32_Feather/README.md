# Projet Bombe - Code/Codes_ESP32/0-Desuet/Desuet_ESP32_Feather

## Description

Ce dossier contient les fichiers de code pour un projet de code ESP32 qui est maintenant obsolète et n'est plus utilisé dans le projet Bombe. Ce projet a été créé avec un ESP32-Feather comme base, mais il a été remplacé par d'autres projets de code qui utilisent des ESP32-WROOM. Les fichiers de ce dossier sont conservés à des fins de référence, mais ils ne sont plus maintenus ni utilisés dans le projet actuel.

## Architecture de ce dossier

Ce dossier contient les fichiers de code pour le projet ESP32-Feather obsolète. Il est organisé de la manière suivante :
- `CircuitPy/` : Contient tous les fichiers de code qui ont été écrits en CircuitPython pour le projet ESP32-Feather. Ces fichiers ne sont plus utilisés, mais ils sont conservés à des fins de référence.
- `Cpp/` : Contient tous les fichiers de code qui ont été écrits en C++ pour le projet ESP32-Feather. Ces fichiers ne sont plus utilisés, mais ils sont conservés à des fins de référence.
