# Projet Bombe - Code/Codes_ESP32/0-Desuet/Desuet_ESP32_Feather/CircuitPy

## Description

Ce dossier contient le projet en CircuitPython pour l'ESP32-Feather qui est maintenant obsolète et n'est plus utilisé dans le projet Bombe. Il se fait envoyer des données en UART par un autre appareil et allume des DELs en fonction de ces données. Ce projet a été remplacé par d'autres projets de code qui utilisent des ESP32-WROOM. Les fichiers de ce dossier sont conservés à des fins de référence, mais ils ne sont plus maintenus ni utilisés dans le projet actuel.

## Architecture de ce dossier

- `lib/` : Contient toutes les bibliothèques utilisées dans le projet CircuitPython pour l'ESP32-Feather. Ces bibliothèques sont nécessaires pour faire fonctionner le code, mais elles ne sont plus utilisées dans le projet actuel.
- `code.py` : Contient le code principal du projet CircuitPython pour l'ESP32-Feather. Ce code reçoit des données en UART et allume des DELs en fonction de ces données. Ce fichier n'est plus utilisé, mais il est conservé à des fins de référence.
