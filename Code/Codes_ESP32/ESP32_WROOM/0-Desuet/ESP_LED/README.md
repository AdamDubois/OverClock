# Projet Bombe - Code/Codes_ESP32/ESP32_WROOM/ESP_LED

## Description

Ce dossier contient les fichiers de code pour le projet ESP32-WROOM qui gère les strips de DELs programmables dans le projet Bombe. Ce projet est actif et maintenu, et est utilisé pour gérer les différentes fonctionnalités liées aux DELs programmables. Le projet reçoit des données en I2C et allume les DELs en fonction de ces données. Ce projet est organisé de manière claire pour faciliter la navigation et la compréhension du code.

## Architecture de ce dossier

Ce dossier contient les fichiers de code pour le projets ESP32-WROOM LED. Il est organisé de la manière suivante :
- `include/` : Contient le fichier de configuration pour les broches utilisées dans le projet ESP32-WROOM qui gère les strips de DELs programmables.
- `src/` : Contient tous les fichiers de code qui ont été écrits en C++ pour le projet ESP32-WROOM qui gère les strips de DELs programmables. Ces fichiers sont actifs et maintenus, et sont utilisés pour gérer les différentes fonctionnalités liées aux DELs programmables dans le projet Bombe.
