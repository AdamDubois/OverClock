# Projet Bombe - Code/Codes_ESP32/ESP32_WROOM/ESP_DEL

## Description

Ce dossier contient les fichiers de code pour le projet ESP32-WROOM qui gère les strips de DELs programmables dans le projet Bombe. Ce projet est actif et maintenu, et est utilisé pour gérer les différentes fonctionnalités liées aux DELs programmables. Le projet allume les strips de DEL sans commandes. Le but est de tester les connexions et le fonctionnement des DELs avant de les intégrer dans le projet global.

## Architecture de ce dossier

Ce dossier contient les fichiers de code pour le projets ESP32-WROOM DEL. Il est organisé de la manière suivante :
- `src/` : Contient tous les fichiers de code qui ont été écrits en C++ pour le projet ESP32-WROOM qui gère les strips de DELs programmables. Ces fichiers sont actifs et maintenus, et sont utilisés pour gérer les différentes fonctionnalités liées aux DELs programmables dans le projet Bombe.
