# Projet Bombe - Code/Codes_ESP32

## Description
Ce dossier contient les fichiers de code pour les différents ESP32 utilisés dans le projet Bombe. Chaque ESP32 a son propre sous-dossier pour organiser les fichiers de manière claire. Les codes sont écrits en C++ et utilisent la plateforme de développement PlatformIO pour faciliter la gestion des dépendances et la compilation.

## Architecture de ce dossier

Ce dossier contient les fichiers de code pour les différents composants du projet Bombe. Il est organisé de la manière suivante :
- `0-Desuet/` : Contient tous les projets de code qui ne sont plus utilisés ou qui sont obsolètes. Ces projets ont été créés avec un autre ESP comme base. Ils sont conservés à des fins de référence, mais ne sont plus maintenus ni utilisés dans le projet actuel.
- `ESP32_WROOM/` : Contient tous les projets de code pour les ESP32-WROOM utilisés dans le projet. Ces projets sont actifs et maintenus, et sont utilisés pour gérer les différentes fonctionnalités du projet Bombe, telles que la gestion des lecteurs de cartes RFID, des switchs, des boutons et des strips de DELs programmables. Chaque projet est organisé de manière claire pour faciliter la navigation et la compréhension du code.
