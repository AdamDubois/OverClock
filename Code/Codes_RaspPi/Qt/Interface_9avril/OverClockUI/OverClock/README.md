# Projet Bombe - Code/Code_RaspPi/Qt/Interface_9avril/OverClockUI/OverClock

## Description 
Ce dossier contient l’ensemble des fichiers utilisés pour la conception et le développement de l’interface.

## Architecture de ce dossier
Ne pas modifier les fichiers autres que ceux spécifiés dans la liste ci-dessous.

- OverClockContent : contient tous les fichiers liés au visuel de l’interface graphique.
- jsonreceive.py : permet d’envoyer les données du jeu vers l’interface.
