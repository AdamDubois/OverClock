# Projet Bombe - Code/Code_RaspPi/Qt/Interface_9avril/OverClockUI/OverClock

## Description
Ce dossier contient l’ensemble des fichiers utilisés pour la conception visuelle de l’interface.

## Architecture de ce dossier
- 3d : fichier 3d pour le projet
- fonts : pour le téléchargement de fonts
- images : dossier regroupant les images uilisées pour l'interface.
- App.qml : programme permettant d'afficher la bonne interface pendant le jeu.
- Enigme1.qml : interface pour l'énigme 1
- Enigme2.qml : interface pour l'énigme 2
- Enigme3.qml : interface pour l'énigme 3
- FinScreen.qml : interface lors de la fin du jeu
- IdleScreen.qml : interface lorsque le jeu est inactif
