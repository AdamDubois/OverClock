# Projet Bombe - Code/Codes_RaspPi/9Avril

## Description

Ici se trouvent les fichiers de code utilisés par le Raspberry Pi pour la journée entrepreneuriale du 9 avril. Ces codes sont fait pour fonctionner rapidement pour être utilisé lors de la journée entrepreneuriale. Les enigmes seront peut-être différentes de celles finales. Après la journée entrepreneuriale, ces codes seront peut-être modifiés pour s'adapter à la version finale du projet Bombe ou seront déplacées dans le dossier désuet du projet Bombe.

## Architecture de ce dossier

Ce dossier contient les fichiers de code pour la journée entrepreneuriale du 9 avril, notamment pour les interactions entre le Raspberry Pi et les ESP32 du projet Bombe. Il est organisé de la manière suivante :
- `RFID/` : Dossier contenant les codes pour l'énigme des lecteurs RFID. Il contient les codes pour lire les données des lecteurs RFID connectés aux ESP32 via I2C et pour envoyer des commandes I2C aux ESP32_DEL pour tester les animations de DEL programmables.
- `Switch/` : Dossier contenant les codes pour l'énigme des interrupteurs. Il contient les codes pour lire les données des interrupteurs connectés aux ESP32 via I2C et pour envoyer des commandes I2C aux ESP32_DEL pour tester les animations de DEL programmables.
